1 - O PHP possui formatos distintos de arquivos para objetivos específicos. Quais são eles?
	- R: .php (Arquivo padrão de programa PHP), .class.php (Arquivo contendo casses PHP), .ic.php (Arquivo de inclusão apenas), .php3 e .php4 (Específicos das versões 3 e 4) e o .phtml (Arquivo contendo programa em PHP e HTML juntos).

2 - Quais as 3 camadas básicas de sistemas web?
	- R: Apresentação, lógica e camada de fonte e gerenciamento de dados.

3 - Qual acamada que tem por objetivo a exibição dos dados e fazer a interface entre o sistema e o usuário?
	- R: Camada de Apresentação

4 - Projetos web trabalham com protocolos de comunicação. Dentre eles, alguns são os mais requisitados como HTTP, FTP, POP. Qual o significado e a função de cada um deles?
	- R: HTTP (Hypertext Transfer Protocol – Responsavel pelo tratamento de pedidos e respostas entre clientes e servidores na World Wide Web), FTP (File Transfer Protocl – Define a forma como os dados devem ser transferidos em uma rede TCP/IP) e POP (Post Office Protocol – Acesso remoto a uma caixa de e-mail).

5 - O PHP possui vários comandos de saída (impressão). Descreva cada um deles, destacando o que os diferencia.
	- R: echo (Comando de impressão padrão para variáveis, tanto para o console como em programas PHP, não retorna nada), print (Usado para imprimir uma String, sempre retorna o valor inteiro 1), var_dump (Usado para imprimir variávei de forma mais explicativa, comum para se realizar debug. Serve para objetos, vetores e etc), print_r (Semelhante ao var_dump, mas seu resultado aparece de forma menos detalhada).
